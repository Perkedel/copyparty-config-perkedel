# Select one of the config!

Here are serveral conf files you can use your copyparty to.

each of these are bespoke config file designed for different device configurations. 
Simply select one of them & it'll choose optimized optons for you

```sh
cd .. # go back to the root folder of this repository, then
./copyparty-sfx.py -c "conf/select-one-of-these-confs.conf"
```
